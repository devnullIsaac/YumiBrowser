# Pull requests are not accepted

**Yumi Browser is a deliberately solo-developer project.** To preserve a
coherent long-term vision and the project's 20-year stability horizon,
external pull requests are not accepted at this time.

Please **close this PR** and instead:

- File a [bug report](../../issues/new?template=bug_report.yaml) if you
  have found a defect, or
- File a [feature request](../../issues/new?template=feature_request.yaml)
  if you have a suggestion, or
- For security-sensitive findings, follow [`SECURITY.md`](../SECURITY.md)
  and report privately.

A well-formed bug report is, in practice, the highest-leverage way to
help this project today.

The full rationale is in [`CONTRIBUTING.md`](../CONTRIBUTING.md). Yumi
Browser is licensed under **AGPL-3.0**, so you remain fully free to fork
the project and pursue your changes in your own tree.

Thank you for your understanding.
